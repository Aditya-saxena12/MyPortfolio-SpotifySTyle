# 🎙️ Stylish Spotify Portfolio — Aditya Saxena

> **"Code as Rhythm, Systems as Symphony."**

A premium, interactive developer portfolio built with **React** and **Vite**, designed with the iconic Spotify aesthetic. This project transforms the traditional resume into an immersive "Now Playing" experience, blending high-end UI design with functional audio components.

## ✨ Key Features

### 🎵 Immersive Music Player
*   **Functional Playback**: Integrated music player with local MP3 support.
*   **Interactive Controls**: Play, pause, skip, and seek functionality with a real-time progress bar.
*   **HMR Support**: Seamlessly transitions between tracks with metadata updates.

### 🎨 Premium UI/UX
*   **Spotify Geometric Language**: Uses circular avatars, pill-shaped buttons, and dark charcoal cards.
*   **Aesthetic Palette**: A sophisticated blend of **Off-white (#F5F5F0)**, **Dark Charcoal (#191414)**, and the classic **Spotify Green (#1DB954)**.
*   **Custom Interactions**: 
    *   Dynamic **Green Dot Cursor** that reacts to clicks.
    *   Smooth **Fade-in-up** scroll animations.
    *   Interactive **Genre Pill Grid** for skills.

### 🗂 Systematic Sections
*   **🏠 Home (Now Playing)**: A hero banner styled as a track overview with quick CTA links.
*   **👤 About (Artist Bio)**: A detailed look at the engineer behind the code.
*   **💼 Experience (Discography)**: Professional journey displayed as a chronological album list.
*   **🚀 Projects (Featured Playlists)**: A visual grid of technical achievements.
*   **📄 Credits (Publications)**: Academic and research highlights.

## 🛠 Tech Stack

*   **Frontend Library**: [React.js](https://reactjs.org/)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Styling**: Pure CSS (Vanilla)
*   **State Management**: React Hooks (useState, useEffect, useRef)
*   **Animation**: CSS Keyframes + Intersection Observer API

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aditya-saxena12/spotify-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd spotify-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
